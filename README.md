# LifeLine Track - Emergency Ambulance Service

A comprehensive web application for emergency ambulance tracking and management with multi-language support and responsive design.

## Features

- **Real-time Ambulance Tracking**: Live GPS tracking with interactive maps
- **Multi-language Support**: Available in English, Hindi, Gujarati, and Marathi
- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Driver Dashboard**: Real-time status updates and navigation
- **Admin Panel**: Fleet management and monitoring
- **Hospital Integration**: Partner hospital information and directions

## Getting Started

### Prerequisites
- Modern web browser with JavaScript enabled
- Internet connection for map functionality

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Hareii1/lifelinee.git
cd lifelinee
```

2. Start a local server:
```bash
python3 -m http.server 8000
```

3. Open your browser and navigate to `http://localhost:8000`

## Features Overview

### Multi-Language Support
- Switch between English, Hindi, Gujarati, and Marathi
- Language preference saved in browser
- Complete UI translation including dynamic content

### Responsive Design
- Mobile-first approach
- Breakpoints: 480px, 600px, 768px, 900px, 991px, 1200px, 1600px

- Hospital finder with directions
